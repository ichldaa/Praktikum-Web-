4.1 Embedded Style Sheet

CSS yang ditulis langsung di dalam file HTML, biasanya diletakkan di tag 'STYLE' pada bagian 'HEAD'. Cocok buat styling halaman tunggal.

4.2 Inline Style Sheet

CSS yang ditulis langsung di dalam elemen HTML pakai atribut style. Praktis tapi kurang rapi kalau untuk halaman besar

4.3 Linked Style Sheet

CSS yang dipisah di file khusus .css lalu dihubungkan ke HTML dengan 'link'. Ini cara paling rapi dan sering dipakai.

4.4 Selector

Digunakan untuk memilih elemen HTML yang mau dikasih gaya. Bisa berupa element selector (p), class selector (.kelas), atau ID selector (#id).

4.5 Grouping Element

Menggabungkan beberapa selector sekaligus biar hemat kode. Misal h1, h2, p {color: blue;} artinya semua h1, h2, dan p jadi biru

4.6 Float

Properti CSS buat ngatur posisi elemen agar bisa "mengapung" di kiri atau kanan, biasanya dipakai buat layout dan wrap teks di sekitar gambar.

4.7 Dropdown Menu

Menu yang bisa muncul/turun saat diarahkan kursor atau diklik. Biasanya dibuat dengan kombinasi HTML 'ul' 'li' dan CSS hover/JavaScript
